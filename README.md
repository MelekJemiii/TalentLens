# 🔍 TalentLens – AI Resume Intelligence Platform

TalentLens is an AI-powered resume analysis platform that evaluates CVs, extracts key skills, and provides actionable feedback to help users improve their employability. It also generates professional downloadable reports with modern design.

---

## 🚀 Live Features

- 📄 Smart resume analysis using AI
- 🧠 Automatic skill extraction
- 📊 Overall resume scoring (0–100)
- 💪 Strengths identification
- ⚠️ Weakness & improvement suggestions
- 🎯 Job match analysis (optional job description)
- 📥 Downloadable professional PDF report
- 🎨 Modern UI with responsive design
- ⚡ Fast and interactive experience

## 🛠️ Tech Stack

**Frontend**
- Next.js 15 (App Router)
- React 18
- Tailwind CSS

**UI & Icons**
- Lucide React
- Custom design system

**PDF Generation**
- jsPDF
- html2canvas

**State Management**
- React Hooks (useState, useEffect)

**Storage**
- sessionStorage (temporary analysis storage)

---

## 📂 Project Structure


/app
/analyze → Resume upload & processing page
/results → AI analysis results (TalentLens core UI)
/components → Reusable UI components
/public → Static assets (logo, screenshots)
/styles → Global styles (Tailwind)
/lib → Helper functions (AI logic, utils)

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/MelekJemiii/talentlens.git
cd talentlens
2. Install dependencies
npm install
3. Run development server
npm run dev
```
## 🚀 Run the Project

Then open:

http://localhost:3000
## 🏗️ Build for Production

```bash
npm run build
npm start
```
